# 🌐 Portfolio Website
## 📌 Project Overview

This project is a **personal portfolio website** designed to present my skills, projects, and evolution as a developer in a clear and engaging way.

The focus was to create a **modern, responsive, and interactive experience**, using only **vanilla web technologies** — reinforcing strong front-end fundamentals.

---

### ⚙️ Implementation

Key technical decisions:

- **Semantic HTML5** for accessibility and structure  
- **CSS3 (Flexbox + Grid)** for layout and responsiveness  
- **Vanilla JavaScript** for performance and control  
- `IntersectionObserver` for scroll-based animations  
- Custom animations (typing effect, hover interactions)  

---

### 📈 Results

- Improved visual hierarchy and readability  
- Smooth navigation and user interaction  
- Lightweight and fast performance  
- Strong demonstration of front-end fundamentals  

---

## ✨ Features

- 🎨 Custom UI with modern design  
- ⌨️ Typing animation in the hero section  
- 🎞️ Scroll-triggered animations  
- 🃏 Interactive hover effects  
- 📱 Fully responsive layout  
- 🔝 Back-to-top button  
- ⚡ No frameworks — pure HTML, CSS, JS
